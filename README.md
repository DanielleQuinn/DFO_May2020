# DFO_May2020
This repository contains the materials for the Software Carpentry DFO workshop series, taking place May 26 - 29


Data Sources  

The Kings Creek Fish Surveys data used in this workshop is a modified version of the publicly avilable data cited below. It is used only as a teaching tool and the modifications and/or use of this data for the purpose of this workshop is not representative of the original data or its intentions.  

Gido, K. B. 2018. CFP01 Fish population on selected watersheds at Konza Prairie . Environmental Data Initiative. http://dx.doi.org/10.6073/pasta/e1ac8845d032315df70819ece0d37c42.
